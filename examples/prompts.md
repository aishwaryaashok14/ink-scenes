# Prompt Examples

Copy any of these straight into your agent. Each one is a technical concept — the skill invents the everyday scene, places the accent, and draws one wordless image.

## Draw one concept

```text
Use $ink-scenes to draw a wordless concept illustration for:

compaction

Everyday scene metaphor, warm cream background, fine ink linework,
one red-orange accent. No words anywhere.
```

## A transformation (input becomes output)

```text
Use $ink-scenes to draw an image for:

embedding — meaning turned into coordinates

One strange but plausible analog machine with a visible in-state and
out-state. Wordless, one dominant accent carrier.
```

## A contrast (before/after)

```text
Use $ink-scenes to draw a two-panel contrast for:

refactoring — same behavior, cleaner insides

Both panels in one canvas, separated by cream space, no divider line.
The panels should rhyme: same angle, same scale.
```

## A scale encounter

```text
Use $ink-scenes to draw:

a context window — how much the model can hold in mind at once

One large patient figure or structure, one small human, lots of cream sky.
```

## A relationship (one thing serving another)

```text
Use $ink-scenes to draw:

a webhook — being notified instead of having to keep checking

Two parties and an exchange through a physical interface — a hatch,
a counter, a mail slot. Wordless, one dominant accent carrier.
```

## Several concepts at once

```text
Use $ink-scenes to draw one image each for: queue, job, cache.

One composed image per concept, each with its own fresh everyday
metaphor — not a collage, not a shared diagram.
```

## Swap the signature color

```text
Use $ink-scenes to draw "vector database", but swap the red-orange accent
for deep teal — same restraint, one dominant carrier, hushed earth tones.
```

## Edit an image

```text
Use $ink-scenes to edit this image:
a readable word appeared on the chalkboard — replace it with illegible
scribble marks, keep everything else identical.
```
